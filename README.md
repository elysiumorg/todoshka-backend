# Todoshka backend
## Description

Backend of todoshka written on TypeScript with using [Nest](https://github.com/nestjs/nest).

## Stack

- [Nest](https://github.com/nestjs/nest)
- TypeScript
- Class-validator
- Class-transformer
- Mongoose
- JWT
## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run dev

# debug mode
$ npm run debug

# production mode
$ npm run prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
